## 1. Project Overview

**SenangStart Actions** is a lightweight, declarative UI framework designed for humans and AI agents. It provides a directive-based reactive system similar to Alpine.js/Vue.js but built with modern standards and modularity in mind.

### Core Philosophy
- **Declarative**: behavior is defined in HTML attributes (`ss-*`).
- **Reactive**: Uses proxies for fine-grained reactivity.
- **Modular**: Directives can be used individually or as a complete bundle.
- **AI-Friendly**: Codebase is structured to be easily parsable and understandable by LLMs.

## 2. Technology Stack

- **Language**: JavaScript (ES Modules).
- **Build System**: Rollup.
- **Testing**: Vitest + JSDOM.
- **Documentation**: VitePress.
- **Package Manager**: npm.

## 5. Development Guidelines

### Coding Standards
- **ES Modules**: Use `import`/`export` syntax.
- **Formatting**: Keep code clean and consistent. 
- **Naming**: camelCase for variables/functions, PascalCase for classes.

### Testing
- **Requirement**: ALL new features must include unit tests.
- **Tool**: Vitest.
- **Coverage**: Aim for high coverage, especially for core logic.
- **Command**: `npm run test` or `npm run test:watch`.

### Building
- The build system is modular.
- **Core**: `senangstart-actions-core.js`
- **Directives**: `senangstart-actions-[name].js`
- **Main**: `senangstart-actions.js` (includes typically used directives)

### Adding Features
1. **Directives**: Add implementation in `src/directives/`, entry in `src/entries/`, and register in `rollup.config.js`.
2. **Core**: Modify `src/core/` for fundamental changes.
